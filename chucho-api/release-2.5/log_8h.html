Detailed Description

Logging macros functions that use the Chucho C API.

These macros handle the machinery of filtering messages by level and adding markers.

Log an event every N times.

The level must be one of TRACE, DEBUG, INFO, WARN, ERROR or FATAL. This will behave exactly as the corresponding macro, like CHUCHO_C_INFO, does with the level replaced.

Parameters
lvlthe level, one of TRACE, DEBUG, INFO, WARN ERROR or FATAL
nthe count after which the event should be logged
lgthe logger, which can either be a char* or a chucho_logger_t*
...printf-style parameters that must include the format text first
