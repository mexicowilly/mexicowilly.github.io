Detailed Description

An abstract representation of an object that can be used to create a real one.

Mementos are used by the configuration system to create configurable objects. You only need to worry about mementos if you create custom types that you would like to make configurable. As a configuration file is read by the configurator, the key-value pairs that it encounters in the file are fed to a collection of mementos that gather information about objects that will be created. Once the memento's realization is complete during configuration, a configurable_factory is used to create a real object using its memento.
