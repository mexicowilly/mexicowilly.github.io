Detailed Description

Configure chucho.

A configurator is used to configure chucho using an input stream. Configuration by default happens automatically when the first logger is requested. A configurator is created that can perform its duties with the information at hand about the configuration file, or lack thereof.

Add a factory.

Any class that expects to be instantiated during configuration time must register itself here with its factory. All predefined chucho configurable classes automatically do this. You only need to call this method if you have a non-chucho-defined type that you would like to be configurable.

Parameters
namethe name as it will appear in the configuration file
factthe factory that will be used to manage creation of the class in question
