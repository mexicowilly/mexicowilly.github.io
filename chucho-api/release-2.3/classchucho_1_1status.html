Detailed Description

Chucho reports what is happening internally using status messages.

These can be monitored by querying the status_manager or by installing a custom status_observer.

enum chucho::status::level
strong

The level of this status event.

These definitions have a trailing underscore because they can conflict with preprocessor macros defined by Windows.

Enumerator
INFO_ 

Information only.

Nothing is wrong.

WARNING_ 

Something went wrong, but not badly enough to inhibit functionality.

ERROR_ 

Chucho is prevented from doing something that it should be doing.
