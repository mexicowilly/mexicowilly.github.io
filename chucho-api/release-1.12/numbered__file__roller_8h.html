Detailed Description

Functions related to numbered file rollers.

A file_roller that numbers the rolled files. This roller can be used to created sequences of numbered files, like my.log, my.log.1, my.log.2, etc....

Create a numbered file roller.

Postcondition
Ownership of the rlr parameter is transferred to the caller, and it must be released with the chucho_release_file_roller() function.
Ownership of the cmp parameter is transferred to the callee.
Parameters

[in]min_indexthe minimum index at which to start numbering files
[in]max_indexthe maximum index at which to stop numbering file
[in]cmpthe file compressor, which may be NULL if no compression is desired
