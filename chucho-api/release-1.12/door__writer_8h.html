Detailed Description

Write to Solaris doors.

Doors are a lightweight form of IPC, kind of like RPC but good. The door to which Chucho writes must be attached to the file system, which is accomplished by calling fattach after creating the door.

The data sent is encapsulated in the chucho_door_event structure from door_event.h.
