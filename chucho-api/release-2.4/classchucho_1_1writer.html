Detailed Description

A writer writes events to whatever destination it is designed to use.

A logger can have as many writers attached to it as it needs. Writers also provide a further means of filtering events, as each writer has a collection of filters. The filters are visited in the order in which they are added.

◆ write()

void chucho::writer::write ( const event &  evt)

Write an event.

This non-virtual method takes care of all the common housekeeping that writers must undertake when writing an event. A check is made to see whether the current thread is already writing something, the presence of the formatter is verified, and the filters are evaluated. If the event can actually be written, then the virtual method write_impl is called.

Parameters
evtthe event to write
