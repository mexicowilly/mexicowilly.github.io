Detailed Description

A serializer that creates a Cap'n Proto message.

The definintion of the message is in the file serialization/chucho.capnp. All of the event data, including the formatted message, will appear in the resulting message. If you only need the formatted message, then please consider using the formatted_message_serializer instead.
