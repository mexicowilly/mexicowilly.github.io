◆ endl()

std::ostream& chucho::endl ( std::ostream &  ls)

An I/O manipulator to write a line ending to a log_stream.

This differs from std::endl in that a platform-appropriate line ending is always written and the stream is not flushed.

There is no need to call this function directly. Typical usage is:

chucho::log_stream stream(lgr);
CHUCHO_M(stream) << chucho::debug << "My dog has fleas." << chucho::endl << chucho::endm;
Deprecated:
There should be no need to use this manipulator. The newline can be inserted into the message as it normally would with the %n pattern_formatter token.

◆ set_level()

set_level_type chucho::set_level ( std::shared_ptr< level >  lvl)
inline

An I/O manipulator to set the level of a log_stream.

All levels that ship with Chucho have their own manipulators, so this manipulator is only required for custom levels. Typical usage is:

chucho::log_stream stream(lgr);
CHUCHO_M(stream) << chucho::set_level(chucho::level::INFO_()) << "My dog has fleas." << chucho::endm;
