Create a RabbitMQ writer.

Postcondition
Ownership of the wrt parameter is transferred to the caller, and must be released with the chucho_release_writer() function.
Ownership of the fmt parameter is transferred to the callee.
Ownership of the ser parameter is transferred to the callee.
Ownership of the cmp parameter is transferred to the callee.
Note
The cmp parameter may be NULL.
The routing_key parameter may be NULL.
Parameters
[out]wrtthe RabbitMQ writer to create
[in]fmtthe formatter
[in]serthe serializer
[in]coalesce_maxThe maximum number of log events to hold in cache before sending them. If -1, then the default of 25 will be used.
[in]cmpthe compressor, which may be NULL
[in]urlthe URL of the RabbitMQ instance
[in]exchangethe exchange to which to post messages
[in]routing_keythe routing key, which may be NULL
Returns
a value from return_code.h indicating success or failure
