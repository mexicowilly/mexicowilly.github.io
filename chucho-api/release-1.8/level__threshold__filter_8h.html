Detailed Description

Functions related to level threshold filters.

A filter that tests the log level. The exact same check on the level that is performed by a logger when it is determining whether an event is filtered or not is performed by this filter. This allows further refinement. A logger can have one writer filtering at one level and another writer filtering at another level.
