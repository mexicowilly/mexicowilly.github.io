void chucho::finalize ( )

Finalize the Chucho library.

This function should be called if you believe your memory leak analysis tools when they tell you that static objects that live the life of the application are memory leaks. You should also use this function if you are using Chucho as a shared object or within an enclosing shared object.

Included by <chucho/finalize.hpp>.

Note
No Chucho APIs may be used after calling this function. The result is extremely undefined.
