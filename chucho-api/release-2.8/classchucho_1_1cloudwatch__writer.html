Detailed Description

Write to AWS Cloudwatch logs.

The events are sent to Cloudwatch in batches of up to 10,000.

If you don't provide a region, then one will be supplied by either the environment variable, AWS_REGION, or the ~/.aws/config file. If the region cannot be gleaned from either of those sources, then it will be empty.
