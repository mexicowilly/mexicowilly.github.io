Chucho

Welcome

Chucho is a logging framework based on the popular Java library, Logback. Chucho makes it easy to add logging to an application with, aside from the log statements themselves, very little extra code having to be written. In most cases you would just need to set the name of the Chucho configuration file, and you would be ready to go.

This is the documentation of the Chucho API. For access to releases, documentation, tickets and source code, please visit the project site.

Attention
The configuration file is your primary interface to the library.

Chucho uses the same architecture as Logback, which involes loggers, writers, formatters and other elements. For more information on the design and use of Chucho, please visit the user manual, which is under construction.

C API

Though the core of Chucho is written in C++14, a wrapping C API is also exposed. This allows Chucho to be used in C programs and in C++ programs that may be built with a compiler that cannot build Chucho, such as a non-C++14 compiler.

The C API is expressed in header files with the extension .h, while the core Chucho C++11 API is in files with the extension .hpp.

Some documentation for the C API is copied directly from the C++ documentation, so you may find reference to C++ classes in the C documentation.
