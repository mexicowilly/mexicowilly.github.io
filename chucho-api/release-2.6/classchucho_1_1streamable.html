Detailed Description

template<typename type>
class chucho::streamable< type >

A class that inherits from streamable gets a logger and a log_stream for free.

In addition to being able to write log events with the LGBL-tagged macros that are available to descendants of loggable, CHUCHO_MS can be used to write to the log_stream that is provided. So, in non-static methods of classes that inherit from streamable, the following may be used:

CHUCHO_INFO_LGBL("All the " << animals << " like me");

And:

CHUCHO_MS << chucho::info << "All the " << animals << " like me" << chucho::endm;

Both will have identical results.

overrideprotectedvirtual

Rename the logger.

This method is meant to be used in cases where a subclass of a subclass of loggable needs a different logger name. Say you have a class hierarchy built off the abstract class shape. You make shape a subclass of loggable, so that everyone gets a logger. Subclasses of shape don't want their loggers to be named shape, so they rename the logger in the subclass' constructor.
