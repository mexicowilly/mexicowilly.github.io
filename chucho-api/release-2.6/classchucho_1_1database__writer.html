Detailed Description

A writer that writes to a database.

Supported databases include any that the SOCI library supports. As of this writing these are Oracle, MySQL, DB2, PostgreSQL, sqlite3, ODBC and Firebird. You must have the back-end client libraries installed, as well as a SOCI library in order to use this writer.

No compile-time linkage is necessary with the back-end SOCI library or the database client libraries, but at run-time, these libraries must be available.

The text of the connection parameter depends on which back-end is being requested. But, generally, it takes the form:

<back-end type>://<space-separated parameters>

For PostgreSQL, this might look like:

postgresql://host=lotsofdoggies.com port=5432 dbname=chucho user=scrumpy password=dumpy

And, for Oracle, it might be something like:

oracle://service=//host.stuff.com:1521/CHUCHO user=scrumpy password=dumpy

Please consult the SOCI documentation for the details of the desired back-end.
