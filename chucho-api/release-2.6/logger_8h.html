◆ chucho_logger_t

typedef struct chucho_logger_t chucho_logger_t

A logger.

This type can be used with the functions chucho_get_logger and chucho_release_logger in order to make logging slightly more efficient, if and only if your compiler supports the _Generic keyword. In this case, you may use the logging macros with either a logger or the name of a logger as the lgr parameter. If your compiler does not support _Generic, then you may only use the name of the logger in the logging macros, so using this type will be nearly pointless.
