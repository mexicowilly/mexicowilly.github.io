Detailed Description

Functions related to Ruby evaluator filters.

Filter by evaulating a Ruby expression. The data carried by the event are available as Ruby global variables, therefore they must be reference with a prefix of '$'. The variables that can be referenced are the following.

VariableTypeMeaning
loggerString The name of the logger
levelString The name of the level
messageString The formatted message
timestampTime The time
file_nameString The file name
line_numberFixnum The line number
functionString The name of the function
markerString The marker or nil if there is no marker

The Ruby expression must return a true or false value.
