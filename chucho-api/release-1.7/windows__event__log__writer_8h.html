)

Create a Windows event log writer.

Postcondition
Ownership of the wrt parameter is transferred to the caller, and it must be released with the chucho_release_writer() function.
Ownership of the fmt parameter is transferred to the callee.
Note
The host parameter can be NULL if connection to the local host is desired.
Parameters
[out]wrtthe writer to create
[in]fmtthe formatter
[in]logthe name of the log to which to write
[in]sourcethe name of the event source
[in]hostthe name of the host to which to connect, which can be NULL to connect to the local host
Returns
a value from return_code.h indicating success or failure
