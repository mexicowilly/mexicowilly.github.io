Detailed Description

A writer that writes to syslog.

If a syslog_writer is given a host name, then it will use UDP to write to the syslog. Otherwise, if the platform supports it, then the writer will use the native syslog interface and allow the system to handle the IPC.

Construct a syslog_writer.

If this constructor is used, then the messages will be written to the local host using the syslog interface, which allows the system to manage the IPC. UDP over a socket is not used in this case.

If the system does not support the syslog C interface, then UDP over a socket is used to connect to the local host.
