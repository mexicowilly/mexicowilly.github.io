Detailed Description

A log event.

You don't need to worry about this. It is used internally, but required by classes that have a public interface.

Construct an event.

The host name and thread ID parameters are what set this constructor apart from the others. This constructor is used by the remote logging server to store the name of the host from which the event came. It is not used when logging is done locally.
