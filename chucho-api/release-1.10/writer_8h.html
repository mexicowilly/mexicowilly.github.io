chucho_rc chucho_wrt_get_filters ( const chucho_writer *  wrt,
chucho_filter **  buf,
size_t  buf_size,
size_t *  count 
)

Retrieve all filters.

An array of filters is provided, which will be filled. If the number of existing filters in the writer exceeds the size indicated by the buf_size paramter, then CHUCHO_INSUFFICIENT_BUFFER is returned. The number of existing filters is always return in the count parameters. You may pass NULL for the buf parameters if you just want to find out how many filters there are.

Postcondition
The ownership of all filters in the returned array is transferred to the caller, and they must be releaesd with the chucho_release_filter() function.
Parameters
[in]wrtthe writer
[out]bufthe array to receive the filters
[in]buf_sizethe size of the array
[out]countthe actual number of filters in the writer
Returns
a value from return_code.h indicating success or failure
