Detailed Description

Functions for ZeroMQ writers.

A writer that publishes ZeroMQ messages. To read log events published to the given endpoint, you would connect a ZeroMQ socket of type ZMQ_SUB. Additionally, a "topic" can be used by providing a prefix in the writer's constructor. If a prefix is provided, then each event will be published as a two-part message.
