Macro Definition Documentation

#define CHUCHO_MS   CHUCHO_M(get_log_stream())

Prepare the internal stream to write a message.

This macro puts the location information of the message, file, line number and function name, into the stream. If this information is not required, then the macro is not necessary. Typical usage is:

1 CHUCHO_MS << chucho::info << "My dog has fleas." << chucho::endm;
Note
This macro may only be used in non-static methods of a class that inherits from streamable.
